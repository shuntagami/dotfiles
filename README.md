# dotfiles
Simple dotfiles includes .zsh with Prezto, .vimrc with dein.vim

## how to use
$ git clone https://github.com/shuntagami/dotfiles.git <br>
$ cd dotfiles <br>
$ chmod +x install.sh <br>
$ ./install.sh <br>
$ zsh <br>
